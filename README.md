# 🎉 EventVista AI - Smart Event Planner

An AI-powered event planning platform built with Streamlit and Groq AI.

## Features
- 🏠 Dashboard with charts and metrics
- ➕ Add events with AI suggestions
- 📋 Manage events and tasks
- 📅 Event calendar & timeline
- 🤖 AI Assistant powered by Groq (LLaMA 3.3)

## Setup
1. Clone the repo
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Create a `.env` file:
   ```
   GROQ_API_KEY=your_groq_api_key
   ```
4. Run the app:
   ```
   streamlit run eventplanner.py
   ```

## Tech Stack
- Streamlit
- Groq AI (LLaMA 3.3 70B)
- SQLite
- Plotly
- Pandas
